# IIS Scanner

A lightweight Python tool for scanning **Microsoft IIS servers** for common security misconfigurations and vulnerabilities.

## 🚀 Features
- Inspects server headers
- Detects common IIS misconfigurations
- Identifies potentially vulnerable endpoints
- Checks for exposed files and directories
- Provides basic recommendations
- Simple and fast scanning process
- Easy to extend with additional checks

## 📦 Requirements
- Python 3
- [shortscan](https://github.com/bitquark/shortscan) (optional)

## ⚙️ Usage
`python iis_scanner.py http://example.com`

## ⚠️ Disclaimer

This tool is intended for educational purposes and authorized security testing only.
Do not use this tool against systems without proper permission.
